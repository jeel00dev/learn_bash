| Command      | Meaning                                             |
| ------------ | --------------------------------------------------- |
| `rm file`    | Deletes immediately (no warning)                    |
| `rm -i file` | Ask before deleting                                 |
| `rm -f file` | Force delete (no questions, even errors ignored ⚠️) |
| `rm -r dir`  | Delete directories recursively                      |
| `rm -ri dir` | Ask before deleting each file in directory          |
